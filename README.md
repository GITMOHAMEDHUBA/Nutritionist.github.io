# أروى بكر  صفحة هبوط احترافية

هذا المشروع عبارة عن صفحة هبوط (Landing Page) ثابتة تم تطويرها باستخدام **HTML، CSS، وJavaScript** بهدف تقديم الهوية الرقمية لأخصائية تغذية علاجية بطريقة احترافية وعصرية. صُممت الصفحة وفق مبادئ **Call-To-Action (CTA)** للتركيز على تحويل الزائر إلى عميل محتمل من خلال عرض الخدمات، إبراز المؤهلات والخبرات، بناء الثقة، وتوجيه المستخدم نحو اتخاذ الإجراء المطلوب عبر التواصل المباشر. كما تتميز بتصميم متجاوب مع مختلف الأجهزة، وتجربة مستخدم سلسة، وهيكل بصري حديث يركز على الوضوح وسهولة الاستخدام.
